# 海报设计 AI 技能包 - 单文件版本

这是一个**超级简化**的版本，只需将 `poster-skills-bundle.json` 文件拖拽到 Claude Desktop 窗口即可使用！

---

## 🚀 3 秒安装方法

### 方法一：直接拖拽（最简单）

1. 打开 Claude Desktop
2. 将 `poster-skills-bundle.json` 文件拖拽到 Claude 窗口
3. 完成！立即可以使用

### 方法二：复制粘贴

1. 打开 `poster-skills-bundle.json` 文件
2. 全选并复制所有内容
3. 粘贴到 Claude Desktop 窗口
4. 完成！

---

## 💡 使用方法

安装后，直接在 Claude 中输入：

### Midjourney 技能
```
使用 Midjourney 主海报生成技能，生成一个未来科技峰会海报提示词
```

### NanoBanana 技能
```
使用 NanoBanana 系列海报生成技能，生成虚拟主播 IP 宣传系列
```

---

## 📦 包含的 8 个技能

### Midjourney 平台
- 🎨 主海报生成
- 🎭 艺术风格匹配
- 📐 构图色彩优化
- 📦 系列海报批量

### NanoBanana 平台
- 👤 主海报生成（角色一致性）
- 🎭 艺术风格匹配（IP 定位）
- 📐 构图优化（角色姿态）
- 📦 系列生成（强制一致性）
