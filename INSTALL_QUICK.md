# 🚀 快速安装指南

## 最简单的安装方式（推荐）✨

只需一条命令即可安装所有 8 个技能！

### 步骤 1：运行安装命令

打开终端，输入：

```bash
claude mcp install https://github.com/scudong/poster-design-ai-skills
```

### 步骤 2：重启 Claude Desktop

完全退出 Claude Desktop 并重新启动。

### 步骤 3：开始使用

在 Claude Desktop 中输入：

```
使用 Midjourney 主海报生成技能，生成一个未来科技峰会海报
```

**完成！** 🎉

---

## 其他安装方式

### 方式二：拖拽安装（无需命令行）

1. 打开 Claude Desktop
2. 将 `poster-skills-bundle.json` 文件拖拽到 Claude 窗口
3. 完成！

### 方式三：克隆项目安装

```bash
# 克隆项目
git clone https://github.com/scudong/poster-design-ai-skills.git
cd poster-design-ai-skills

# 安装技能
npm run install:skills

# 验证安装
npm run validate
```
