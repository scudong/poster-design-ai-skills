# 🎨 海报设计 AI 技能包 - 拖拽即用版本

## 🚀 超简单安装方法

### 方法一：macOS 一键安装（最简单）✨

只需一条命令：

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/scudong/poster-design-ai-skills/main/install-macos.sh)"
```

**完成！** 重启 Claude Desktop 即可使用。

### 方法二：直接拖拽

1. **打开 Claude Desktop**
2. **将 `poster-skills-bundle.json` 文件拖拽到 Claude 窗口**
3. **完成！立即可以使用**
