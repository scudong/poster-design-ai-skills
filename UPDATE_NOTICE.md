## 🚀 当前推荐的安装方式

### macOS 一键安装（最简单）✨

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/scudong/poster-design-ai-skills/main/install-macos.sh)"
```

### 拖拽安装（无需命令行）

1. 打开 Claude Desktop
2. 将 `poster-skills-bundle.json` 拖拽到窗口
3. 完成！

### 手动配置

查看 [INSTALL.md](INSTALL.md) 了解详细步骤。
